## Usage

Just import the module and you'll benefit from an augmented, and typed, `BigNumber` class!

```typescript
import "ethers-maths";

const WAD = BigNumber.pow10(18);

BigNumber.from(1).wadMul(WAD); // 1
BigNumber.from(WAD.mul(2)).rayMul(0.5e27); // WAD
```
